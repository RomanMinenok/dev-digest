# Design

Reference design material (mockups, screenshots, notes) collected by the
[`spec-creator`](../.claude/agents/spec-creator.md) subagent while writing a
spec in [`/specs`](../specs/README.md). Organized as
`design/<feature-slug>/`, one folder per feature. Not hand-maintained —
files here are working references for a spec, not a source of truth on
their own.
